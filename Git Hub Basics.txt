0. Setup repository 
	git clone <repository URL>

1. Change directiory to repository
	cd <respository directory>

2. Check status of repository
	git status

3. Add updated files
	git add filename	# Add a specified file
	git add -A		# Add a all files

4. Add a commit message
	git commit -m "<commit message>"
	
	if you hit commit without a message press "esc" and type ":wq"
5. Push to github
	git push

6. Pull from github
	git pull

7. Get possible git commands
	git
